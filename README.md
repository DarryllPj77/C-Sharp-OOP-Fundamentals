Encapsulation
=============

Encapsulation is the bundling of data and methods into a single unit while hiding
the internal details from the outside world.

Projects in this section:
  -- Encapsulation: Basic encapsulation with private fields and public methods
  -- Encapsulationv2: Enhanced version with property validation
  -- Encapsulationv3: Advanced encapsulation patterns
  -- Encapsulationv4: Real-world encapsulation scenarios
  -- Encapsulationv5: Best practices in encapsulation
  -- Encapsulationv6: Method-based encapsulation (no public properties)
  -- Encapsulationv7: Hybrid approach combining methods and properties
  -- IntroEncapsulation: Introduction to encapsulation concepts

Key Concepts Covered:
  -- Private fields and public accessors
  -- Property getters and setters
  -- Data validation and constraints
  -- Information hiding principles
  -- Access modifiers (private, public, protected)


Constructors
============

Constructors are special methods used to initialize objects when they are created.

Projects in this section:
  -- IntroConstructors: Introduction to constructors and object initialization

Key Concepts Covered:
  -- Constructor syntax and declaration
  -- Default and parameterized constructors
  -- Constructor overloading
  -- Object initialization patterns


Classes and Objects

Understanding the fundamentals of classes and objects in C#.

Projects in this section:

  -- IntroClass: Deep dive into class structure and usage

Key Concepts Covered:
  -- Class declaration and structure
  -- Creating objects and instances
  -- Instance variables and methods

Getting Started
===============

Prerequisites:
  -- .NET 8.0 SDK or later
  -- Visual Studio 2022 or VS Code
  -- Basic knowledge of C#

Setup:
  1. Clone this repository
  2. Navigate to the OOPIntro solution
  3. Open OOPIntro.sln in Visual Studio
  4. Build the solution (Ctrl+Shift+B)
  5. Run individual projects to see examples


Project Structure
=================

Each project is organized as follows:

  OOPIntro/

Running the Examples
====================

Using Visual Studio:
  1. Open the solution file
  2. Right-click a project
  3. Select "Set as Startup Project"
  4. Press F5 to run

Using .NET CLI:
  1. Open terminal in project directory
  2. Run: dotnet run
  3. View console output
